# catalyst

## Style Guide

Classes use camel-case and start with upper-case: `MyClass`
Structs use camel-case and start with lower-case: `myStruct`
Non-static Functions use camel-case and start with lower-case: `myFunction`
Static Functions use camel-case and start with upper-case: `MyStaticFunction`
Non-member Variables use snake-case and start with lower-case: `my_variable`
Member Varaibles use camel-case and start with upper-case: `MyMemberVariable`
Constants use all upper-case separated by underscores: `MY_CONSTANT`
Macros use all upper-case separated by underscores: `MY_MACRO`
